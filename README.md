# DES Supplementary files

Supplementary code, logs, and analysis files supporting the DES study on _Achromobacter ruhlandii_ evolution, adaptation, and epidemiology.

## BactDating RDS 

BactDating output containing dated phylogeny and model parameters

## BEAST:
- Constant population size model
  - XML configuration file
  - Log files from 3 independent MCMC runs
- Relaxed clock model
  - XML configuration file
  - Log files from 2 independent MCMC runs
- Skygrid model
  - XML configuration file
  - Log files from 3 independent MCMC runs


### Notes:
- XML files define model parameters, priors, and MCMC settings used in BEAST analyses

- Log files contain parameter traces for convergence assessment (e.g., ESS, posterior distributions)

- Multiple independent runs were performed to ensure convergence and consistency
